Refactor the searchAndWithdrawByName method (lines 602-683) to use verified item search:

**Current problem:** Uses fixed pixel offset (bankBounds.x + 90, bankBounds.y + 110) and taps blindly.

**New approach:**
1. After typing search query and waiting, get the item ID from ItemManager using the item name
2. Use findAndVerifyItem(script, itemId) to get verified bounds for the item
3. If findAndVerifyItem returns null, use searchBankForItem(script, itemId) to scroll and find
4. If found, tap the verified bounds with the withdraw action
5. Clear search and return result

**Key changes:**
- Add item ID lookup: `int itemId = script.getItemManager().getItemId(itemName)` - if this doesn't exist, search all ItemIDs by name match
- Replace the fixed-offset tap loop with:
  ```java
  // find item with verification
  Rectangle itemBounds = findAndVerifyItem(script, itemId);
  if (itemBounds == null) {
      // scroll search as fallback
      itemBounds = searchBankForItem(script, itemId);
  }
  if (itemBounds == null) {
      // not found
      clearSearch(script);
      return false;
  }
  // tap verified bounds
  boolean success = script.getFinger().tap(itemBounds, new String[]{action});
  ```

**Note:** The ItemManager may not have getItemId(String name). If not available, we need an alternative:
- Option A: Look up ItemID constant by reflection (unreliable)
- Option B: Type search, then use findItemInVisibleBank with a searchable item created from the visual area
- Option C: Skip itemId-based verification and rely on menu name matching via tapGetResponse

**Recommended approach (Option C):** Since we're searching by name, modify the flow:
1. Type search query
2. Scroll to top of filtered results
3. Use tapGetResponse on the first visible item slot to check if it matches
4. If matches, tap with withdraw action
5. If not, scroll and repeat

This avoids needing itemId lookup entirely while still verifying the item.

```java
public static boolean searchAndWithdrawByName(Script script, String itemName, int amount) {
    // existing pre-condition checks...

    // type search query (existing)
    if (!typeSearch(script, itemName)) {
        return false;
    }

    // wait for filter
    script.pollFramesHuman(() -> false, script.random(400, 600));

    // determine withdraw action (existing)
    String action = (amount == 0) ? "Withdraw-All" :
                    (amount == 1) ? "Withdraw-1" : "Withdraw-" + amount;

    // scroll to top first
    BankScrollUtils.scrollToTopWithCheck(script, 20);
    script.pollFramesHuman(() -> false, script.random(200, 400));

    // get bank bounds
    Rectangle bankBounds = script.getWidgetManager().getBank().getBounds();
    if (bankBounds == null) {
        clearSearch(script);
        return false;
    }

    // search with verification
    for (int scrollCount = 0; scrollCount <= MAX_SCROLL_ITERATIONS; scrollCount++) {
        // scan visible bank area for matching item
        Rectangle foundBounds = findMatchingItemByName(script, itemName);

        if (foundBounds != null) {
            // found verified item - withdraw
            boolean success = script.getFinger().tap(foundBounds, new String[]{action});
            if (success) {
                script.pollFramesHuman(() -> false, script.random(200, 400));
                clearSearch(script);
                return true;
            }
        }

        // check if at bottom
        if (BankScrollUtils.isAtBottom(script)) {
            break;
        }

        // scroll down
        BankScrollUtils.scrollDown(script);
        script.pollFramesHuman(() -> false, script.random(200, 400));
    }

    clearSearch(script);
    return false;
}
```

```java
/**
 * Finds an item in visible bank by name match using tapGetResponse.
 * Scans item slots and verifies menu entity name matches expected item.
 */
private static Rectangle findMatchingItemByName(Script script, String itemName) {
    Rectangle bankBounds = script.getWidgetManager().getBank().getBounds();
    if (bankBounds == null) return null;

    // bank item grid - each slot is roughly 36x32 pixels
    // grid starts at offset from bank bounds
    int startX = bankBounds.x + 70;
    int startY = bankBounds.y + 90;
    int slotWidth = 48;  // includes padding
    int slotHeight = 36;
    int cols = 8;  // bank has 8 columns
    int rows = 6;  // visible rows

    for (int row = 0; row < rows; row++) {
        for (int col = 0; col < cols; col++) {
            int x = startX + (col * slotWidth) + (slotWidth / 2);
            int y = startY + (row * slotHeight) + (slotHeight / 2);

            Rectangle slotBounds = new Rectangle(
                startX + (col * slotWidth),
                startY + (row * slotHeight),
                slotWidth - 4,
                slotHeight - 4
            );

            // tap to get menu response
            MenuEntry entry = script.getFinger().tapGetResponse(true, slotBounds);
            if (entry == null) continue;

            // check if entity name contains our search term
            String entityName = entry.getEntityName();
            if (entityName != null && entityName.toLowerCase().contains(itemName.toLowerCase())) {
                script.log(BankSearchUtils.class, "found matching item: " + entityName);
                return slotBounds;
            }
        }
    }

    return null;
}
```

```java
private static Rectangle findMatchingItemByName(Script script, String itemName) {
    Rectangle bankBounds = script.getWidgetManager().getBank().getBounds();
    if (bankBounds == null) return null;

    // first item slot position (bank search consolidates results to top-left)
    int x = bankBounds.x + 90;
    int y = bankBounds.y + 110;
    Rectangle firstSlot = new Rectangle(x - 15, y - 15, 36, 32);

    // verify via menu
    MenuEntry entry = script.getFinger().tapGetResponse(true, firstSlot);
    if (entry == null) return null;

    String entityName = entry.getEntityName();
    if (entityName != null && entityName.toLowerCase().contains(itemName.toLowerCase())) {
        script.log(BankSearchUtils.class, "verified item at first slot: " + entityName);
        return firstSlot;
    }

    script.log(BankSearchUtils.class, "first slot does not match: expected [" + itemName + "] got [" + entityName + "]");
    return null;
}
```
